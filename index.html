8/5/2024 - ALPS Post-Workshop Reflection (Revised 8/13/2024)

Creating Artificial Intelligence which can think like humans has become a hot topic in recent years. Modern engineers have attempted to construct AI using the technique of machine learning: “teaching” machines to find patterns in past experiences and use them to generate new results. But while machine learning models like ChatGPT boast impressive capabilities, they lack the fundamental step in the actual human thought process: reason.

ChatGPT's hallucinations are a humorous defect in the model which demonstrates its lack of reasoning. The model only regurgitates text which it has seen before, without knowing if what it produces is right or wrong. This lack of reason proposes a serious concern about the usage of such AI: how can we trust machines which don’t understand their own conclusions?

However, a different type of AI addresses this problem: “explainable machines”. This summer, I spent two months at the University of Texas at Dallas (UTD) studying machines which, instead of relying on pattern matching, actually utilized logic to derive conclusions. Instead of just mimicking conclusions humans have produced, explainable machines mimic the process of how humans produce logical conclusions.

We can equip explainable machines with logical reasoning by formalizing the thought process into programmatic “rules”. We can then tell machines to follow these rules when looking at information given to them to derive conclusions. By following this process, we can see how the machine comes to the conclusion that it produces. In class, we looked at a project attempting to automate Canadian law using explainable machines. Explainable machines could have serious real-world impact in fields where transparency is critical in decision-making.

To build explainable machines, we used the logic programming language of Prolog. This language was structured perfectly to design explainable machines: programmers, instead of telling the program what to do, would define facts and rules, telling the machine how it should reason with information provided. The machine would then follow those logical rules to derive a solution for the problem defined by the programmer. Now, the only task for programmers would be to define in rules the problem they were trying to solve, and then the machine could solve it for them. We wrote Prolog code to solve famous puzzles including n-queens, tower-of-hanoi, and send+more=money, and I also wrote a Wordle solver. Code I wrote can be found here: https://github.com/AlxV07/UTD_ALPS_2024

My experience this summer at UTD was very inspiring. I enjoyed learning about the Prolog language and how to formalize the human thought process in code. It was especially informative to learn about explainable machines and how they could fill the gap in reason of pure machine learning models. Instead of creating artificiality intelligent machines, we could create actually intelligent machines, utilizing logic and reason to advance sensible decision-making in the world.

1/30/2024 - I rebranded (5->7)

Kind of note-worthy news! I changed one character in my username everywhere! From "AlxV05" to "AlxV07", links & profiles which could be updated have been updated. Why? It goes back to why I have my username in the first place.

I first found out I needed my own username ~5 years ago (10 yrs old) while making some random account I don't have access to anymore. I came up with "Alx" quick enough, decided I liked version numbers so added the "V0" soon after, and finally I just had to choose the last character. 5 sounded cool ("al-ks-vee-oh-five"). Nothing special about 5; just sounded cool.

Well I'm mature now ;) and I'd prefer to have my markers have more meaning than just it "sounding cool". So I looked at my username: Alx still fits; three-char compact name-like tags are always nice to have around. Second part: the version identifier - I think I'll keep it, I'm getting updated :D So that means I need to choose a number greater than 5. 7's the only single-digit number greater than 0 that has two syllables (unique). I'm also Catholic, and the number 7 symbolises some significant topics: perfection, the 7 days of creation, the 7 cardinal virtues, and many more references from the Bible.

After updating everything it feels kinda nice. It's always good to have a switch-up every now and then (as long as it isn't too drastic nor does it cause any problems). Well, I guess till next time. Bye!
